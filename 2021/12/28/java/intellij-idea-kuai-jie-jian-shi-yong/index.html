工具用得好，下班下得早

Ctrl + Alt + T

使用if，if/else, while, do/while, for, try/catch, try/finally, try/catch/finally, synchronized, Runnalbe 包围代码块

Alt + Enter

代码快速补全、包引入、临时变量生成等

Ctrl + Alt + M

将代码块提取到一个方法

Ctrl + Shift + 上下方向键

上下移动单行代码和代码块

Class.new + （Alt + Enter）

Class.new创建一个Class类型的匿名对象，Alt + Enter将匿名对象赋值给一个新建的变量

instance.try

根据instance实例对象快速生成try/catch代码块
